# yt-dlp-ando-script
Personal yt-dlp script for easier downloading of my format.

Use the .py file for now.
The .bat file is inconsistent with handling the new yt-dlp.


# Requirements
```sh
pip install pyyaml
```

# yt-dlp Download
I am using version `yt-dlp 2025.10.22`

[Source File](https://github.com/yt-dlp/yt-dlp/releases)

[Download Here](https://github.com/yt-dlp/yt-dlp/releases/download/2025.10.22/yt-dlp_win.zip)

Download the `yt-dlp_win.zip` file

### How to unzip:

Place the `.zip` file in main_path
Check `config.yaml` file for this reference


# Config File
Edit `config.yaml` file for changing the directories of your files.
